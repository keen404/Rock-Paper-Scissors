1 = กระดาษ (paper)

หากผู้เล่นแพ้ จะประกาศว่าผู้เล่นแพ้ พร้อมบอกเหตุผล เช่น กระดาษชนะฆ้อน

หากผู้เล่นชนะ จะเพิ่มคะแนนไปยังตัวแปรผู้เล่น 1 แต้ม
